Future Citizen System / UI Concept

FC Wallet
主权身份与可信交互移动端界面概念稿

这不是一款传统的加密钱包首页方案，而是一套以身份、信任、治理为中心的数字公民操作界面。整个视觉方向延续 Future Citizen 蓝图中的机构感、深海蓝网格、冷光结构线与权威金色节点，同时把链上复杂度隐藏在更平静、更可信的交互框架里。

Identity First Trust Visible Stablecoin Native Citizen + Organization Private by Design
Design Direction

第一版界面重点

1. 首页不是余额页

先展示身份、凭证与待处理的可信请求，再展示资产和支付入口。

2. 支付体验接近 Web2

用户默认看到稳定币金额和支付结果，Gas 兑换逻辑藏在回执与详情里。

3. 企业能力是正统模块

组织工作台、多签审批和收益分润不做成附属插件，而是完整入口。

4. 安全与恢复可理解

MPC、安全设备、卡片关系和恢复状态都用用户可理解的方式表达。

Palette / Tone

视觉语言与设计 Tokens

采用深蓝基础色、青蓝结构高亮、浅金核心节点强调，配合蓝图式网格、层叠卡片与证书感信息容器，形成“数字制度基础设施”的产品气质。

Foundation Navy

系统背景、主界面空间、结构基底。

Signal Cyan

可验证状态、系统线条、关键交互反馈。

Authority Gold

身份锚点、权威节点、重要确认动作。

Institution Glass

信息卡片、叠层容器、收纳复杂数据。
